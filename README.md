# OOP Assignment

This repository contains a basic Object Oriented Programming assignment in Python.

## Files
- encapsulation.py – Demonstrates encapsulation using a BankAccount class.
- inheritance.py – Demonstrates inheritance using base and derived classes.
- polymorphism.py – Demonstrates polymorphism using method overriding.

## How to run
Make sure Python 3 is installed, then run:

python encapsulation.py  
python inheritance.py  
python polymorphism.py
